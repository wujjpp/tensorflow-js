# Tensorflow.js 演示样例

## How to Install

```shell
$ git clone https://github.com/wujjpp/tensorflow-js.git
$ cd tensorflow-js
$ npm install
```

## How to Run and Build

### Run

```shell
$ npm start
 ```

### Build

```shell
$ npm run build
```

### Run in dist

```shell
$ npm run start:dist
```

## How to Update

```shell
$ git checkout master
$ git pull origin master
$ npm install
```
